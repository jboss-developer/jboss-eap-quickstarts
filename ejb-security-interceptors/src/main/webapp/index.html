ejb-security-interceptors Quickstart

What is it?

The ejb-security-interceptors quickstart demonstrates demonstrates how to use client and server side interceptors to switch the identity for an EJB call in Red Hat JBoss Enterprise Application Platform.

Access the Application

This example application is not intended to be opened in a browser. Instead, you execute the client using the mvn exec:exec command.

Additional Information

Additional information can be found in the README file at the root of this quickstart directory.
