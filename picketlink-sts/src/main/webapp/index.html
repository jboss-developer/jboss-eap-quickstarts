picketlink-sts Quickstart

The picketlink-sts quickstart demonstrates how to deploy a fully compliant WS-Trust Security Token Service (STS) to Red Hat JBoss Enterprise Application Platform.

There is no user interface for this quickstart. Instead, follow these steps to test the service.

  1. Right-click on the `jboss-picketlink-sts` project and choose `Run As` --> `Maven Build`.
  2. Enter "picketlink-sts" for the `Name`.
  3. Enter "exec:java" for the `Goals:`.
  4. Click `Run`.
  5. Review the output in the console window.
