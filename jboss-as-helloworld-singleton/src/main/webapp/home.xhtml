Helloworld EJB 3.1 Singleton Quickstart

This example demonstrates a singleton session bean that maintains state for 2 variables: a and b.

A counter is incremented when you click on the link to the variable name. If you close and restart your browser, or if you have multiple browsers, you can see that the counter always increments the last value. These values are maintained until you restart the application. To test the singleton bean, click on either "Increment" button below.
